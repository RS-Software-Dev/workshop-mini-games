# The Simple Rules

<v-clicks>

Every cell follows these rules each turn:

1. **Lonely** ğŸ˜¢ - Dies if it has less than 2 neighbors
2. **Happy** ğŸ˜Š - Survives if it has 2 or 3 neighbors  
3. **Crowded** ğŸ˜µ - Dies if it has more than 3 neighbors
4. **Birth** ğŸ£ - Empty cell becomes alive with exactly 3 neighbors
